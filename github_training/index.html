Phase 3: Git Command

These are some examples of git command

Add new file to Staging Area

git add {file}

Commit changes

git commit -m {message}

Push changes to remote repository

git push {remote} {remote branch}

Example: You have a change in file Card.jsx and wanna push this to remote

                    
# add file to Index
git add Card.jsx 
# commit changes that have been added to index with message
git commit -m "Update Card Component: [Update color]"
# push code to branch main in remote origin
git push origin main
